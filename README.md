# Where She Stands

A single-page interactive data story comparing gender equity across 6 industries — exposing gaps in pay, leadership, family leave, and career growth — layered by race and ethnicity.

🔗 [Live Site](https://where-she-stands.vercel.app/)

## About

Women are told to "negotiate better" and "lean in" — but no one hands them a map. **Where She Stands** gives women the map. It compares 6 industries across 4 dimensions of gender equity, layered by race and ethnicity, and driven by what the user actually cares about most. The story doesn't celebrate progress. It exposes gaps, names who is being left behind, and ends with a clear, personalized answer: *here is where you stand the best chance.*

## Features

- **6 Industries Compared** — side-by-side equity data across pay, leadership, family leave, and career growth
- **Race & Ethnicity Filter** — intersectional layer revealing how outcomes differ within gender
- **Personalized Recommendation Engine** — users rank what matters most to receive a tailored industry match
- **Interactive Data Visualizations** — D3.js-powered charts that respond to scroll and user input
- **Real Stories** — narrative personas grounding the data in lived experience
- **Responsive Design** — mobile-first layout with glassmorphic cards and a bold editorial palette

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no framework)
- [D3.js v7](https://d3js.org/) for data visualization
- Google Fonts — Playfair Display (serif headlines) + DM Sans (body) + Barlow Condensed (data labels)
- GitHub Pages for hosting
